# React Split Button

The React [`Split Button`](https://react-api.syncfusion.com/split-button/overview) is a graphical user interface component that displays Split Button with visual severity levels. It differentiates Buttons with icons and colors to denote the importance and context of the Button. It has several built-in features, such as severities, variants, icons, and a template.

### WAI-ARIA attributes

The [`Split Button`](https://react-api.syncfusion.com/split-button/overview) component followed the [WAI-ARIA](https://www.w3.org/WAI/ARIA/apg/patterns/alert/) patterns to meet the accessibility. The following ARIA attributes are used in the Splitbutton component:

| Attributes | Purpose |
| --- | --- |
| <mark>role</mark> | Indicates the Split Button component as button, Split Button popup as menu, and the Split Button popup action items as menuitem. |
| <mark>aria-haspopup</mark> | Indicates the availability and type of interactive Split Button popup element. |
| <mark>aria-expanded</mark> | Indicates whether the Split Button popup can be expanded or collapsed, as well as indicates whether its current state is expanded or collapsed. |
| <mark>aria-disabled</mark> | Indicates that the element is perceivable but disabled, so it is not editable or otherwise operable. |

### Keyboard interaction

The [`Split Button`](https://react-api.syncfusion.com/split-button/overview) component followed the [keyboard interaction](https://www.w3.org/WAI/ARIA/apg/patterns/alert/#keyboardinteraction) guideline, making it easy for people who use assistive technologies (AT) and those who completely rely on keyboard navigation. The following keyboard shortcuts are supported by the Splitbutton component.

| **Windows** | **Mac** | **To do this** |
| --- | --- | --- |
| <kbd>Space , Enter</kbd>| <kbd>Space , Enter</kbd> | Opens and closes the popup.|
| <kbd>Esc</kbd>| <kbd>Esc</kbd> | Closes the opened popup. |
| <kbd>↑</kbd>| <kbd>↑</kbd> | Navigates up or to the previous action item. |
| <kbd>↓</kbd>| <kbd>↓</kbd> | Navigates down or to the Next action item. |
