# React Column Chart

A column chart visually represents data using vertical bars, making it easy to compare values across different categories or time periods.

## Overview

To visualize data as a column chart, set the [`type`](https://react-api.syncfusion.com/chart/chartseriesprops#type) property to `Column` within the [`ChartSeries`](https://react-api.syncfusion.com/chart/chartseriesprops) component.

## Data label

To display data labels on the column chart, set the [`visible`](https://react-api.syncfusion.com/chart/chartdatalabelprops#visible) property to `true` within the [`ChartDataLabel`](https://react-api.syncfusion.com/chart/chartdatalabelprops) component.

<Preview onDemand={true} reset={true} content={<ColumnLabel />} path="chart/columnLabel.tsx" />

## Column placement

To control the side-by-side arrangement of columns in a chart, set the [`enableSideBySidePlacement`](https://react-api.syncfusion.com/chart/overview#enablesidebysideplacement) property to `false` in the `Chart` component.

- When `false`, columns will overlap if they share the same category axis value.
- When `true` (default), columns are placed side by side for better visual distinction.-by-side arrangement of the columns.

## Customizing column spacing and width

You can control the spacing and width of column series in the chart using the following properties:

- [`columnSpacing`](https://react-api.syncfusion.com/chart/chartseriesprops#columnspacing):

    Defines the space between multiple column series.
    - Accepts values from 0 to 1.
    - A lower value results in tighter spacing, while a higher value increases the gap between columns.

- [`columnWidth`](https://react-api.syncfusion.com/chart/chartseriesprops#columnwidth):

    Determines the width of each individual column.
    - Also accepts values from 0 to 1.
    - A smaller value produces thinner columns, whereas a larger value makes them wider.

These properties allow you to fine-tune the layout and visual balance of your column chart for better readability and presentation.

## Column width in pixels

To define the exact width of columns in pixels, use the [`columnWidthInPixel`](https://react-api.syncfusion.com/chart/chartseriesprops#columnwidthinpixel) property within the `ChartSeries` Component.

- This property allows you to specify a fixed column width, ensuring consistent sizing regardless of chart size or data density.
- It is especially useful when you want precise control over the visual layout of your column chart.

## Series customization

You can customize the appearance of the column series in the `ChartSeries` component by setting the following properties: 

- [`fill`](https://react-api.syncfusion.com/chart/chartseriesprops#fill) – Sets the color of the column in the series.  
- [`opacity`](https://react-api.syncfusion.com/chart/chartseriesprops#opacity) – Controls the transparency level of the column, allowing for varying visibility.  
- [`border`](https://react-api.syncfusion.com/chart/chartseriesprops#border) – Defines the thickness and color of the border around the column.  
- [`cornerRadius`](https://react-api.syncfusion.com/chart/chartseriesprops#cornerradius) – Specifies the radius of the corners of the columns, allowing for rounded edges.
