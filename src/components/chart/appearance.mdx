# Customizing React Chart Appearance

You can enhance the overall look and feel of a chart by configuring the following properties in the `Chart` component:
- [`background`](https://react-api.syncfusion.com/chart/overview#background): Sets the chart’s background using a solid color or an image.
- [`border`](https://react-api.syncfusion.com/chart/overview#border): Defines the border’s color, width, and style to visually frame the chart.
- [`margin`](https://react-api.syncfusion.com/chart/overview#margin): Controls the spacing around the chart, helping with layout and positioning.

## Chart area

The chart area refers to the region where data is plotted, excluding elements such as titles, legends, and axes.
You can customize this area using the [`ChartArea`](https://react-api.syncfusion.com/chart/chartareaprops) component by adjusting the following properties:

- [`background`](https://react-api.syncfusion.com/chart/chartareaprops#background): Sets the background color or image behind the plotted data.
- [`border`](https://react-api.syncfusion.com/chart/chartareaprops#border): Defines the outline around the chart area for better visual separation.
- [`margin`](https://react-api.syncfusion.com/chart/chartareaprops#margin): Manages internal spacing within the chart layout to ensure proper alignment and clarity.

## Chart title

To display a title on the chart, use the [`text`](https://react-api.syncfusion.com/chart/charttitleprops#text) property within the [`ChartTitle`](https://react-api.syncfusion.com/chart/charttitleprops) component.

You can customize the title’s appearance using the following properties:

- [`fontSize`](https://react-api.syncfusion.com/chart/charttitleprops#fontsize) – Sets the font size of the title text.
- [`background`](https://react-api.syncfusion.com/chart/charttitleprops#background) – Defines the background color behind the title.
- [`color`](https://react-api.syncfusion.com/chart/charttitleprops#color) – Specifies the color of the title text.
- [`position`](https://react-api.syncfusion.com/chart/charttitleprops#position) – Determines the placement of the title (e.g., `top`, `bottom`).
- [`align`](https://react-api.syncfusion.com/chart/charttitleprops#align) – Aligns the title text (e.g., `center`, `left`).
- [`border`](https://react-api.syncfusion.com/chart/charttitleprops#border) – Adds a border around the title for visual emphasis.

## Chart subtitle

Use the [`text`](https://react-api.syncfusion.com/chart/charttitleprops#text) property in the [`ChartSubtitle`](https://react-api.syncfusion.com/chart/charttitleprops) component to display a subtitle below the main chart title.

You can customize the subtitle’s appearance using the following properties:

- [`fontSize`](https://react-api.syncfusion.com/chart/charttitleprops#fontsize) – Sets the font size of the subtitle.
- [`background`](https://react-api.syncfusion.com/chart/charttitleprops#background) – Defines the background color.
- [`color`](https://react-api.syncfusion.com/chart/charttitleprops#color) – Specifies the text color.
- [`position`](https://react-api.syncfusion.com/chart/charttitleprops#position) – Determines the placement relative to the chart.
- [`align`](https://react-api.syncfusion.com/chart/charttitleprops#align) – Aligns the subtitle text (e.g., `left`, `center`).
- [`border`](https://react-api.syncfusion.com/chart/charttitleprops#border) – Adds a border around the subtitle for visual emphasis
