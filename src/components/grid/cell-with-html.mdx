# Displaying HTML Content Cells in React Data Grid

The SyncfusionÂ® React Data Grid component supports rendering HTML elements, such as images, links, and formatted text, in both header and data cells to present rich content with custom layouts. By default, HTML content is encoded to prevent security risks. To render raw HTML directly, set the [`disableHtmlEncode`](https://react-api.syncfusion.com/data-grid/columnprops#disablehtmlencode) property to `false`, allowing formatted elements to display as intended within cells or headers.
