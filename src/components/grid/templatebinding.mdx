# Empty Record Template in React Data Grid

The [`emptyRecordTemplate`](https://react-api.syncfusion.com/data-grid/overview#emptyrecordtemplate) property allows to display custom content when the grid has no data. This replaces the default string message with elements such as text, images, or components.
