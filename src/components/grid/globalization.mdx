# Localization in React Data Grid

The SyncfusionÂ® React Data Grid component includes a built-in Localization library to customize displayed text for different languages and cultural preferences. The static text on various elements can be adapted to languages such as "Arabic", "German", "French", and more by setting the locale property in a custom React `Provider` with a translation object.

## Configuration steps

1. Choose the desired [`locale`](https://github.com/syncfusion/react-locale) for the application, as provided in the Syncfusion React localization resources.
2. Install the `CLDR` (Unicode Common Locale Data Repository) package to enable culture specific formatting:
    
    <Snippet content={[{ title: "npm", code: `npm install react-cldr-data` }]} />

3. Import the `L10n` class and `loadCldr` function from the [`@syncfusion/react-base`](https://www.npmjs.com/package/@syncfusion/react-base) package.
4. Import the required localization files for the chosen language and the necessary `CLDR` data files:
  
    <Snippet content={[ { title: "App.tsx", code: `import { L10n, loadCldr } from "@syncfusion/react-base";
    import Localization from './locale.json';
    import * as enAllData from '@syncfusion/react-cldr-data/main/en/all.json';
    import * as deAllData from '@syncfusion/react-cldr-data/main/de/all.json';` } ]} />

5. Call the `loadCldr` function with the imported data to enable formatting and parsing for the selected culture:

   <Snippet content={[ { title: "App.tsx", code: `loadCldr(enAllData, deAllData);` } ]} />

6. Use the `L10n.load` method to load your translation object.
7. Configure the `locale` property in the `Provider` to set the selected language.

**Customizing locale text**

To customize locale text by defining a translation object with the required key value pairs. Load the translation object using the `L10n.load` method and configure the `locale` property in the custom React `Provider` to apply the customized text.

<Snippet content={[ { title: "App.tsx", code: `import { L10n } from '@syncfusion/react-base';
// Load translation object for custom locale.
L10n.load({
  'en-custom': {
    grid: {
      emptyRecord: 'Nothing found!',
      add: 'Create New',
      edit: 'Modify',
      delete: 'Remove',
      update: 'Save Changes',
      cancel: 'Discard'
    }
  }
});
<Provider locale={'en-custom'} >
<Grid dataSource={orderData} />
</Provider>
` } ]} />
