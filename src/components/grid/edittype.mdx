# Edit Types in React Data Grid
 
The Syncfusion® React Data Grid component uses the [`edit.type`](https://react-api.syncfusion.com/data-grid/columneditconfig#type) property to assign editors based on column data types, streamlining input handling and customization. Default editors are automatically applied for common data types such as text, number, date, and boolean. The supported edit types are listed below.

|Edit Type Value | Description |
|-----------------|-------------|
| `EditType.TextBox` | Renders a `TextBox` component for string data type columns. |
| `EditType.NumericTextBox` | Renders a `NumericTextBox` component for numeric data types (e.g., integers, doubles, floats, shorts, bytes, longs, long doubles, decimals). |
| `EditType.DropDownList` | Renders a `DropDownList` component for string data type columns with predefined options. |
| `EditType.CheckBox` | Renders a `CheckBox` component for boolean data type columns. |
| `EditType.DatePicker` | Renders a `DatePicker` component for date type columns. |
 
To configure edit types, use the `edit.type` property within the column configuration. This enables customized data input based on column data types.

## TextBox editor

The `EditType.TextBox` type uses the Syncfusion® React [`TextBox`](https://react.syncfusion.com/textbox#getting-started) component for editing string data. The textbox can be customized using the [`edit.params`](https://react-api.syncfusion.com/data-grid/columneditconfig#params) property, which supports attributes such as [`clearButton`](https://react-api.syncfusion.com/textbox/overview#clearbutton), [`placeholder`](https://react-api.syncfusion.com/textbox/overview#placeholder), and other textbox-specific properties.

## NumericTextBox editor
 
The `EditType.NumericTextBox` type uses the Syncfusion® React [`NumericTextBox`](https://react.syncfusion.com/numeric-textbox#getting-started) component for editing numeric data. The numeric textbox can be customized using the [`edit.params`](https://react-api.syncfusion.com/data-grid/columneditconfig#params) property, which supports attributes such as [`decimals`](https://react-api.syncfusion.com/numeric-textbox/overview#decimals), [`format`](https://react-api.syncfusion.com/numeric-textbox/overview#format), [`spinButton`](https://react-api.syncfusion.com/numeric-textbox/overview#spinbutton), and other numeric textbox specific properties.

### Restricting decimal points in NumericTextBox
 
The Data Grid component restricts decimal input in the `NumericTextBox` during editing by using the `edit.params` property. Setting [`validateOnType`](https://react-api.syncfusion.com/numeric-textbox/overview#validateontype) to `true` prevents decimal input while typing, and `decimals` to "0" ensures only whole numbers are displayed. Disabling spin buttons through the `spinButton` property provides additional control. This configuration is applied to the "Price" column for precise numeric input.
