# Grid Lines in React Data Grid

The SyncfusionÂ® React Data Grid component allows configuration of row and column separator visibility through the [`gridLines`](https://react-api.syncfusion.com/data-grid/overview#gridlines) property. Supported values include:

- `Default`: Horizontal lines only. 
- `None`: No lines. 
- `Both`: Horizontal and vertical lines.  
- `Horizontal`: Horizontal lines only.
- `Vertical`: Vertical lines only.
