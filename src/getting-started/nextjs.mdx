# Getting Started with a Next.js App using React Components

This guide walks you through setting up a `Next.js` application and integrating Syncfusion<sup>®</sup> React components.

## What is Next.js?

[Next.js](https://nextjs.org/) is a React framework that simplifies building fast, SEO-friendly, and user-centric web applications. It offers features like server-side rendering, automatic code splitting, routing, and API routes, making it ideal for modern web applications.

## Prerequisites

Before getting started with the `Next.js` application, ensure the following prerequisites are met:

* [Node.js 18.17](https://nodejs.org/en) or higher.

* Compatibility with macOS, Windows, or Linux.
