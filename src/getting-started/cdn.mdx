# Getting Started with Single Page Application

This section guides you through integrating Syncfusion® React components into a browser-based project using a `CDN` services such as [esm.sh](https://esm.sh/), without the need for a build step. This approach is ideal for quick embedding components in environments where bundling tools like `Webpack` or `Vite` are not used.

## What is esm.sh?

`esm.sh` is an ES Module CDN (Content Delivery Network) that converts npm packages into native ES Modules (ESM), It allows you to load Syncfusion<sup>®</sup> React packages directly in the browser, providing a modern approach to using ECMAScript modules in `HTML` documents.
