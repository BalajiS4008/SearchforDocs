# Localization support in React Components

The `Localization` library enables Syncfusion<sup>®</sup> React UI components to adapt to various languages, ensuring accessibility and usability for global audiences beyond English.

## Loading translations

To implement translations, utilize the `load` function and `Provider` from the [@syncfusion/react-base](https://www.npmjs.com/package/@syncfusion/react-base) module and localization texts from the [@syncfusion/react-locale](https://www.npmjs.com/package/@syncfusion/react-locale) package. In the example below, the application’s language is configured to French (`fr`) using the `locale` property of the `Provider`. This configuration ensures that grid elements, such as column headers are rendered in the specified language.

## Changing the current locale

To switch the language of Syncfusion<sup>®</sup> React UI components, update the `locale` property in the `Provider` context to the desired culture name.
