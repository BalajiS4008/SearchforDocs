# React Component Themes

Syncfusion<sup>速</sup> React UI components include a built-in `Material3` theme, offering both light and dark mode options. The `Material3` theme adheres to Material Design 3 standards, ensuring a modern, consistent, and intuitive user experience. Optimized for performance, this theme enables faster app loading while maintaining a polished, elegant appearance with minimal overhead.

## Applying themes to individual components

Syncfusion<sup>速</sup> theming system allows you to apply themes to individual components from their respective packages, enabling granular customization without importing the entire theme bundle.

To reference an individual component from its package:

<Snippet 
  content={[ 
    { title: "SCSS", code: `@import "<dependent-package>/styles/<theme_name>.scss";
@import "react-notifications/styles/message/<theme_name>.scss";` },
    { title: "CSS", code: `@import "<dependent-package>/styles/<theme_name>.css";
@import "react-notifications/styles/message/<theme_name>.css";` }
  ]} 
/>

**Example:**

<Snippet 
  content={[ 
    { title: "SCSS", code: `@import "react-base/styles/material3.scss";
@import "react-notifications/styles/message/material3.scss";` },
    { title: "CSS", code: `@import "react-base/styles/material3.css";
@import "react-notifications/styles/message/material3.css";` }
  ]} 
/>
