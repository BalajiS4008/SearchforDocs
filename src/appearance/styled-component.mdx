## Adding styled component to the application

To style a Syncfusion<sup>速</sup> React component using `styled-components`, follow these steps:

1. Import `styled-components` in the `src/App.tsx` file.

2. Use the <a href="https://styled-components.com/docs/api#styled" target="_blank" rel="noopener noreferrer">styled</a> factory to wrap the Syncfusion速 React component.

3. Override the component's styles to create a styled version of it.
